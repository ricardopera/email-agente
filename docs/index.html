Email Agente

Versão 1.0.0

Download GitHub

Extração automatizada de informações de emails

Extraia dados importantes como números de processos judiciais e valores monetários dos seus emails automaticamente.

Funcionalidades

Conexão IMAP

Conecta-se a servidores de email via IMAP com suporte a SSL e TLS.

Busca Avançada

Localiza emails não lidos com assunto específico.

Suporte a Caracteres Especiais

Reconhecimento completo de acentos e caracteres especiais do português.

Campos Personalizáveis

Defina quais informações deseja extrair dos emails.

Múltiplos Formatos

Suporta extração de texto, números e datas com formatação automática.

Exportação para Excel

Salva os dados extraídos em arquivo Excel com formatação apropriada.

Formatos Suportados

Texto

Extrai texto simples de campos personalizados.

Número

Converte valores do formato brasileiro (1.234,56) para decimal.

Data

Converte datas do formato brasileiro (dd/mm/aaaa) para o formato internacional.

Como Usar

1

Configure sua conta

Preencha os campos de email, senha e servidor IMAP.

2

Defina os campos de extração

Adicione campos personalizados e selecione o formato adequado (texto, número ou data).

3

Teste a conexão

Verifique se as configurações estão corretas usando o botão de teste.

4

Processe os emails

Clique em "Processar Emails" para iniciar a extração de dados.

5

Visualize os resultados

Um arquivo Excel será gerado com todos os dados extraídos.
